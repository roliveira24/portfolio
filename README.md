# Rual's Marketing Portfolio

A modern, responsive marketing portfolio showcasing development and marketing expertise.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Animations

## Project Structure

```
src/
  app/              Next.js app directory
  components/       Reusable React components
  data/             Portfolio data (projects, skills, testimonials)
  styles/           Global styles
public/
  images/           Static assets
```

## Sections

- **Hero** - Eye-catching intro with CTA
- **About** - Bio, skills, and background
- **Projects** - Featured case studies
- **Services** - What you offer
- **Testimonials** - Social proof
- **Contact** - Get in touch

## Customization

Edit `src/data/portfolio.ts` to update your information, projects, and testimonials.

## Deploy

```bash
npm run build
```

Deploy on [Vercel](https://vercel.com) with zero configuration.
